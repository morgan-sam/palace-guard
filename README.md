# 𝕻𝖆𝖑𝖆𝖈𝖊 𝕲𝖚𝖆𝖗𝖉

A small text based roleplay game made using OpenAi's GPT-3.5. 

You are a courier who has to deliver a letter to the king and you must convince a guard to let you enter the castle.
